Challenge 5: Accessibility Statement

Some accessibility concerns that these challenges might present to the viewer include text alternatives and color/color contrast. For example, visual or hearing impairments may stop some users from being able to see or hear some content. After doing some solution research, I learned that this problem can be addressed using the alt attribute, which provides a description of an image and can be picked up by a screenreading in order to be read aloud to the user. With challenge 2 on the other hand, someone who is color blind may not be able to view the content as I intended. One way this can be addressed is by not relying on color alone for information, and to ensure that the website's text color always contrasts well with the background color.
